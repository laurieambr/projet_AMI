# AMI Chat - Client

Interface utilisateur Streamlit pour l'application AMI Chat.

## Installation

1. Installez les dépendances :
```bash
pip install -r requirements.txt
```

## Configuration

Définissez les variables d'environnement :
- `SERVEUR_IP` : Adresse IP du serveur API
- `SERVEUR_PORT` : Port du serveur API

## Lancement

```bash
streamlit run app.py
```

L'interface sera accessible à l'adresse : http://localhost:8501

## Fonctionnalités

- Chat en temps réel avec streaming
- Historique des conversations
- Bouton pour effacer l'historique
- Interface utilisateur intuitive
